## Geld verdienen met Project Fuga

Niels Leenheer is mede oprichter van Salonhub dit zijn kapsalons en een reserveringssysteem. Hij vertelde over project Fuga dit is iets waarme je via het web kassa's en bonnenprinters kan aansturen.
Hij liet een demo zien en het werkte allemaal erg goed. De bonnenprinters werkte via een API waar je de informatie heen moet sturen in een bijzondere taal die nog ouder is dan HTML.
Ascii is de taal van printers.
